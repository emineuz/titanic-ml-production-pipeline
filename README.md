# Titanic Survival Prediction – End-to-End Machine Learning Pipeline

This project demonstrates a production-oriented machine learning workflow built on the Titanic dataset. The pipeline includes exploratory data analysis, feature engineering, model training, calibration, threshold optimization, deployment simulation, monitoring, and drift analysis.

## Project Workflow

1. Exploratory Data Analysis
2. Data Cleaning
3. Feature Engineering
4. Logistic Regression Baseline
5. Random Forest Modeling
6. Hyperparameter Tuning
7. Model Interpretability (SHAP)
8. Probability Calibration
9. Threshold Optimization
10. Cost-Sensitive Decision Policy
11. Deployment Simulation
12. Model Monitoring
13. Data Drift Simulation
14. Concept Drift Simulation

## Key Highlights

- Built a calibrated Random Forest classifier
- Applied cost-sensitive threshold optimization
- Simulated deployment pipeline
- Implemented monitoring and drift detection
- Evaluated robustness under noisy conditions
